# two_dimensions目录：2维水上机器人项目代码

1. 在two_dimensions/config/config.py下配置好目标和机器人的初始2维坐标

2. 执行main.py的主程序。运行日志保存在logs目录下

以目标坐标(5, 3), 初始坐标(0, 0)为例，主程序循环调用chatGPT，机器人执行运行动作后，最终到达目的地。
